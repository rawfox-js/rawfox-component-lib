# Circular-progress

圆形进度

```html
<rf-circular-progress unknown="false" value="0.3"></rf-circular-progress>
```

未知进度

```html
<rf-circular-progress unknown="true"></rf-circular-progress>
```

## 属性

| 名称    | 可选 | 类型    | 默认值 | 介绍               |
| ------- | ---- | ------- | ------ | ------------------ |
| value   | 是   | number  | 0.4    | 进度，应为0到1的数 |
| unknown | 否   | boolean | 无     | 是否为未知进度     |

## 插槽

无
